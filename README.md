# 원티드 프론트엔드 프리온보딩 인턴십 4주차 과제 (개인)

## 과제 목표 
-  주어진 데이터를 기반으로 시계열 차트 만들기
-  마우스 호버 시 id, value_area, value_bar 데이터를 튤팁 형태로 제공
-  필터링 기능 구현
    - 특정 구역 클릭 시 하이라이트 효과
